# Elo Learning Platform

Platformă educațională cu sistem de rating Elo pentru elevi.  
## Funcționalități

- Login / Register
- Sistem de rating Elo
- Exerciții / task-uri
- Leaderboard
- Admin panel (CRUD)
## Tehnologii

- FastAPI
- PostgreSQL
- React
- Docker
## Ideea

Fiecare utilizator are un rating Elo:
- răspuns corect → +rating
- răspuns greșit → -rating
